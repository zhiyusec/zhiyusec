## ⚙️ 数据与智能流向（关键通路）
资产层（Nexus） → 情报层（Orion） → AI层（Omnibrain）
资产变动触发情报分析，AI层生成威胁预测。

代码审计（Sentinel） → 洞察（Insight） → 报告中心（DocGuard）
漏洞数据流入生命周期管理，修复验证后输出报告。

渗透验证（Phantom） ↔ 修复验证（Aegis） ↔ AI层策略优化（Agent）
攻防结果被AI用于优化防御策略，形成“智能防御闭环”。

知识图谱（Horizon） ↔ AI推理（XMind / Oracle）
图谱驱动AI语义推理，实现威胁根因关联与安全知识增强。

全局调度（Atlas / Core） ↔ 多智能体协作（Guardian / Agent）
实现任务级与智能体级双重联动，支持跨模块任务自动化执行。
